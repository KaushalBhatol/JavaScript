## Primative Data Type

1. [String](#string)
2. [Numbers](#numbers)
3. [Boolean](#boolean)
4. [Null](#null)
5. [Undefined](#undefined)

### string

* string is collection of characters.

Syntax

```JavaScript
let name = "Kaushal";
```

### numbers

* JavaScript variables are auto detect type of numbers. no need to spacify.

```JavaScript
let 33;
let 45.5;
```

### boolean

* booleans values are `true` and `false`.

```JavaScript
console.log(isDrive);
```

### null

```JavaScript
let thisNull = null;
```

### undefined

```JavaScript
let myUndefined = undefined;
```
